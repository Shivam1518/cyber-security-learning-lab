# ðŸ”¹ Categories of Log Sources

## 1. **Endpoint Logs**
Collected from:
- Windows Event Viewer  
- Linux Syslog  
- MacOS logs  
- EDR agents  

### Common Windows Logs:
| Event ID | Meaning |
|---------|----------|
| 4624 | Successful login |
| 4625 | Failed login |
| 4688 | Process creation |
| 1102 | Audit logs cleared |
| 4720 | User account created |
| 4728 | Added to admin group |

### Common Linux Logs:
- `/var/log/auth.log`  
- `/var/log/syslog`  
- SSH logs  
- Sudo attempts  
- Cron jobs  

---

## 2. **Network Logs**
Generated by:
- Firewalls  
- Routers  
- Switches  
- IDS/IPS  
- VPN gateways  

### Examples:
- Allowed/blocked connections  
- Port scanning  
- Suspicious outbound traffic  
- Unauthorized internal movement (east-west traffic)  

---

## 3. **Application Logs**
Collected from:
- Web servers (Apache, Nginx, IIS)  
- Databases (MySQL, PostgreSQL)  
- Authentication apps (AD, Okta)  
- Business applications  

### Examples:
- Login failures  
- Unexpected API calls  
- SQL errors  
- File upload attempts  

---

## 4. **Cloud Logs**
From:
- AWS CloudTrail  
- Azure Activity Logs  
- GCP Audit Logs  
- Cloud security tools  

### Examples:
- Console logins  
- Role changes  
- IAM misconfigurations  
- S3 bucket access  
- Serverless activity (Lambda, Functions)  

---

## 5. **Security Tools Logs**
From tools like:
- EDR  
- WAF  
- Antivirus  
- DLP  
- Honeypots  

These provide **enriched, security-focused telemetry**.

---

## 6. **Threat Intelligence Logs**
Includes:
- Malicious IPs  
- Hashes  
- Domains  
- URLs  
- C2 servers  
- IOC feeds  

Used for IOC-based correlation.
